## Methodology

### Scoring Rubric (out of 25)

| Dimension | Max | What it measures |
|-----------|-----|------------------|
| Specificity | 5 | Content depth and length (file size as proxy) |
| Sources | 5 | Metadata source count + inline Sources section + in-content citations |
| Non-repetition | 5 | Content depth via sub-sections (h3 count) |
| Natural tone | 5 | Narrative quality, in-content data attribution |
| Actionable value | 5 | Internal links to related content + structural depth |

### Classification Rules

**KEEP** (not in this list): Articles meeting ANY of:
- 3+ metadata sources AND inline Sources section AND file size >= 6KB
- 3+ metadata sources AND file size >= 15KB
- 2+ metadata sources AND file size >= 30KB AND inline Sources AND 3+ internal links

**ENRICH** (this list): Everything between KEEP and REWRITE

**REWRITE** (not in this list): Articles with file size < 4.8KB

### Data Sources

- Article metadata from `/tmp/article_data_clean.json` (slug, category, sources count, FAQs count)
- File sizes from `src/app/[locale]/learn/_articles/*.tsx`
- Inline Sources section detected via `grep -rl 'Sources</h2>'`
- In-content citations detected via `grep -rli 'according to|study shows|research from|data from'`
- Internal link count via `href="/learn/"` occurrences in each file
- Sub-section count via `h3Class` occurrences in each file
